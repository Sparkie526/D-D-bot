┌─────────────────────────────────────────────────────────────────────────────────────────┐
│ SESSION MANAGEMENT LAYER (Per Guild)                                                   │
├─────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                         │
│  sessions[guildId] = {                                                                  │
│    ┌─ Player Data ─────────────────────────────────────────────────────────────────┐   │
│    │ • players: { userId → characterName }                                         │   │
│    │ • activePlayers: [{ userId, displayName, characterName }, ...]              │   │
│    │ • originalNicknames: { userId → originalName }                              │   │
│    └────────────────────────────────────────────────────────────────────────────────┘   │
│                                                                                         │
│    ┌─ Game State ──────────────────────────────────────────────────────────────────┐   │
│    │ • active: boolean                                                             │   │
│    │ • history: [{ role, content }] (max 20 messages)                             │   │
│    │ • currentLocation: "dungeon|forest|tavern|generic"                           │   │
│    └────────────────────────────────────────────────────────────────────────────────┘   │
│                                                                                         │
│    ┌─ Turn-Taking (Multi-Player) ──────────────────────────────────────────────────┐   │
│    │ • turnOrder: [userId1, userId2, ...]                                         │   │
│    │ • currentTurnIndex: 0                                                         │   │
│    │ • turnTimeoutHandle: 75 second auto-advance timer                            │   │
│    │ • pendingAction: { action, expectedDC } waiting for roll                     │   │
│    │ • lastRollResult: { playerName, dice, total, dc }                            │   │
│    └────────────────────────────────────────────────────────────────────────────────┘   │
│  }                                                                                      │
│                                                                                         │
└─────────────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────────────┐
│ GAMEPLAY ENGINE                                                                        │
├─────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                         │
│  ┌────────────────────────────────────────────────────────────────────────────────┐    │
│  │ ACTION DETECTION & ROLL HANDLING                                               │    │
│  │                                                                                │    │
│  │  detectActionKeywords(action)  →  DEFAULT_DCS  →  DC value                    │    │
│  │                                                                                │    │
│  │  [ACTION: "I attack the goblin"]  →  ["attack"]  →  DC 10                    │    │
│  │                                                                                │    │
│  │  If roll needed:                                                               │    │
│  │    1. Prompt player: "Roll 1d20 to attempt"                                    │    │
│  │    2. Store pending action + expected DC                                       │    │
│  │    3. Wait for /roll response                                                  │    │
│  │    4. Compare: roll total >= DC ? SUCCESS : FAILURE                            │    │
│  │    5. Pass result to LLM with action for context                               │    │
│  └────────────────────────────────────────────────────────────────────────────────┘    │
│                                                                                         │
└─────────────────────────────────────────────────────────────────────────────────────────┘
                                           ↓
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│ LLM INTEGRATION LAYER (OpenAI or Ollama)                                              │
├─────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                         │
│  buildSystemPrompt()                                                                    │
│    ├─ Base DM instructions (personality, rules, turn-taking)                          │
│    ├─ [OPTIONAL] Character sheet data                                                  │
│    ├─ World notes (from worlds/*.txt)                                                  │
│    └─ Turn/narrative guidance                                                          │
│                                                                                         │
│  askDM(guildId, userMessage, playerName)                                               │
│    ├─ Adds message to session.history                                                  │
│    ├─ Constructs: [systemPrompt, ...history]                                          │
│    ├─ Calls OpenAI/Ollama                                                              │
│    ├─ Returns narration (2-4 sentences)                                                │
│    └─ Adds response to history (keeps last 20)                                        │
│                                                                                         │
│  Context Window Budget (4K tokens):                                                    │
│    • System prompt: 400-500 tokens                                                     │
│    • World notes: 100-1000 tokens                                                      │
│    • Chat history: 200-300 tokens                                                      │
│    • Available: ~2700 tokens remaining                                                 │
│                                                                                         │
└─────────────────────────────────────────────────────────────────────────────────────────┘
